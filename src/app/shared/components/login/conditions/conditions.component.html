1. La utilización de este portal es totalmente gratuita. No necesitas una cuenta para poder ver el contenido del mismo.

2. Con una cuenta en tu haber podrás interactuar con el portal, hacer un seguimiento de tu aprendizaje conforme lo vayas utilizando. Tendrás acceso a tu propio perfil el cuál podrás compartir para enseñar a los demás de que estás hecho.

3. Tus datos se usarán exclusivamente para el uso de este portal. En ningún caso se proporcionarán a terceros. Los datos más delicados como las contraseñas están encriptados, solamente tú tendrás acceso a ellos.

4. Recuerda mantener una actitud sana a la hora de interactuar con otros usuarios. Todos queremos aprender. Tú también puedes enseñar algo a los demás.

5. El contenido del portal es extrictamente orientado a un público tecnológico de cualquier nivel, sobretodo centrado en desarrollo web con Javacript. También es un buen lugar para aprender desde cero alguna tecnología web moderna.
