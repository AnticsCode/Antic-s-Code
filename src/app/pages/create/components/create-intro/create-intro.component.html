Presentamos Antic's Code Desktop un programa de escritorio con el cual serás capaz de crear contenido para este mismo portal.

Es más fácil de lo que crees, para ello sigue estos sencillos pasos:

  • Descarga la aplicación para tu plataforma
  • Ingresa con tu misma cuenta de Antic's Code
  • Entra en la sección de Crear, icono de lápiz verde
  • Sigue las instrucciones
  • Una vez aprobado, tu Artículo estará en vivo
